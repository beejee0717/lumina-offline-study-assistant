# Lumina 📸✨

**Lumina** is a high-performance, offline-first study assistant built with Flutter. It transforms physical notes into interactive study materials using on-device OCR and lightweight NLP algorithms.

## 🚀 The Philosophy: "Intelligence without the Bulk"
Unlike modern "AI" apps that rely on heavy 1GB+ Large Language Models (LLMs) or expensive cloud APIs, Lumina uses **Algorithmic Logic** to provide a seamless, private, and ultra-fast experience—all under **50MB**.

## 🧠 Core Features

### 🔍 Smart Scan (OCR)
- Powered by **Google ML Kit**.
- Instantly digitizes handwritten or printed notes with high precision.
- Works 100% offline; your data never leaves your device.

### 📝 Automated Quiz Generation
- **Fill-in-the-Blanks:** Uses a custom **Keyword Extraction Algorithm** to identify key nouns and concepts.
- **Matching Games:** Automatically pairs terms and definitions found in your notes.
- **Distractor Logic:** Intelligently pulls "wrong answers" from surrounding context to create challenging multiple-choice questions.

### ⚡ Professional Study Tools
- **Flashcard Export:** Seamlessly export your data to CSV or Anki.
- **Smart Highlighting:** Identifies core sentences using frequency-based summarization patterns.
- **Native Speech-to-Note:** Dictate notes during lectures using built-in device processing.

## 🛠️ Technical Stack
- **Framework:** Flutter (Android/iOS)
- **OCR:** Google ML Kit Text Recognition
- **Local Database:** Isar/Hive (NoSQL)
- **Engine:** Custom Dart NLP & TF-IDF Logic

## 🛡️ Privacy First
Lumina is built for the privacy-conscious student. 
- **No Internet Required.**
- **No Account Needed.**
- **No Data Tracking.**
